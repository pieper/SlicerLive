All section bodies are real LiveInterface primitives. Open SceneRendering.py:278 placeholder is the landing site. Click cards, drag handles, slide everything — state goes to the console log via change events.

▼ Volume context VolumeContext
Modality + units come from MRML/DICOM (will fall back to a histogram-based guess). Intent shortlists the preset library below.
▼ Presets PresetPicker
Shortlist filtered by Modality + Intent. (Next step: each card's strip becomes a mini-rendering of the current volume with that preset applied, instead of a static gradient.)
▼ Scalar transfer function CombinedTransferFunctionEditor
Drives vtkVolumeProperty.GetRGBTransferFunction() + GetScalarOpacity() → SlicerWGPU lut_array (256×4 RGBA).
▼ Gradient opacity TransferFunctionEditor Enabled
1D curve over gradient magnitude → SlicerWGPU grad_lut_array (256×1 α). Default Glass-style ramp sharpens surfaces without losing internal detail.
▼ Window / Level WindowLevelEditor
Drives SlicerWGPU's clim (scalar range the TF applies across). Drag either edge handle, click the background to snap, or arrow-key for fine control.
▼ Shading PhongShadingPanel
Drives k_ambient / k_diffuse / k_specular / shininess on the ImageField. The preview is a live Phong-shaded sphere using current coefficients + material color.
▼ Lighting LightingPanel
Drag the warm key light + cool fill light around the front hemisphere. Maps to scene-wide light_direction/intensity + fill_light_* uniforms.
